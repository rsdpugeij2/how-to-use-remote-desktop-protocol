# 🍎 Using Remote Desktop Protocol (RDP) on macOS

This guide shows how to connect to a Windows PC using RDP from macOS.

---

## 📌 Requirements

- macOS device (MacBook, iMac, etc.)
- Internet connection
- Access to a Windows PC with RDP enabled

---

## 📥 Step 1: Install RDP Client

Download **Microsoft Remote Desktop** from the Mac App Store.

---

## 🚀 Step 2: Open the App

1. Launch **Microsoft Remote Desktop**
2. Click **Add PC**

---

## 🌐 Step 3: Enter PC Details

- **PC Name:** Enter the IP address or hostname  
- **User Account:** Add your Windows username & password  

---

## 🖥️ Step 4: Connect

1. Double-click the saved PC
2. Accept certificate (if prompted)
3. You are now connected 🎉

---

## ⚙️ Optional Settings

- 🖥️ Display resolution  
- 🔊 Sound settings  
- 📁 Share folders between Mac and Windows  

---

## 🔒 Security Tips

- Use strong passwords 🔐  
- Connect via VPN 🌐  
- Avoid public Wi-Fi  

---

## ❗ Troubleshooting

### Can't connect?
- Check IP address  
- Make sure Windows PC is ON  

### Connection failed?
- Verify RDP is enabled on Windows  
- Check firewall settings  

### Login issues?
- Confirm username format: `PCNAME\username`  

---

## ✅ Done!

You can now connect to a Windows PC from your Mac using RDP 🚀
