# 🍏 Using Remote Desktop Protocol (RDP) on iOS (iPhone & iPad)

This guide shows how to connect to a Windows PC using RDP from an iPhone or iPad.

---

## 📌 Requirements

- iPhone or iPad
- Internet connection
- Windows PC with RDP enabled

---

## 📥 Step 1: Install RDP App

Download **Microsoft Remote Desktop** from the App Store.

---

## 🚀 Step 2: Open the App

1. Launch the app
2. Tap **+ (Add)**
3. Select **Add PC**

---

## 🌐 Step 3: Enter PC Details

- **PC Name:** IP address or hostname  
- **User Account:** Windows username & password  

---

## 🖥️ Step 4: Connect

1. Tap the saved PC
2. Accept certificate (if prompted)
3. You are now connected 🎉

---

## ⚙️ Features

- Touch controls 👆  
- On-screen keyboard ⌨️  
- Zoom and gestures 🤏  
- External keyboard support ⌨️  

---

## 🔒 Security Tips

- Use strong passwords 🔐  
- Avoid public Wi-Fi 🌐  
- Use VPN for secure access 🔑  

---

## ❗ Troubleshooting

### Can't connect?
- Check IP address  
- Ensure Windows PC is ON  

### Connection failed?
- Check firewall & port 3389  

### Login issues?
- Verify username/password  

---

## ✅ Done!

You can now access your Windows PC from iPhone/iPad using RDP 🚀
