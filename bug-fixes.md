# 🐞 Bug Fixes & Common Issues (RDP)

This section helps fix common problems when using Remote Desktop Protocol (RDP).

---

## ❌ Problem: Can't Connect

### ✅ Fix:
- Check if Remote Desktop is enabled on Windows  
- Verify IP address is correct  
- Ensure both devices are on the same network or VPN  

---

## ❌ Problem: Connection Timed Out

### ✅ Fix:
- Check firewall settings  
- Make sure port **3389** is open  
- Restart the target PC  

---

## ❌ Problem: Login Failed

### ✅ Fix:
- Check username and password  
- Try format: `PCNAME\\username`  
- Make sure the user has RDP permission  

---

## ❌ Problem: Black Screen After Login

### ✅ Fix:
- Restart Remote Desktop session  
- Update graphics drivers  
- Try lower resolution  

---

## ❌ Problem: Slow Connection / Lag

### ✅ Fix:
- Reduce display quality  
- Disable background apps  
- Use a faster internet connection  

---

## ❌ Problem: Remote Desktop Not Available

### ✅ Fix:
- Ensure Windows edition supports RDP (Pro/Enterprise)  
- Enable Remote Desktop in settings  

---

## 💡 Tips

- Always keep your system updated 🔄  
- Use VPN for secure and stable connection 🔐
