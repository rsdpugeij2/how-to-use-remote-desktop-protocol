# 🪟 Using Remote Desktop Protocol (RDP) on Windows

This guide explains how to enable and use Remote Desktop on Windows step by step.

---

## 📌 Requirements

- Windows 10/11 Pro, Enterprise, or Server (Host PC)
- Internet or local network connection
- Administrator access

---

## 🚀 Step 1: Enable Remote Desktop

1. Open **Settings**
2. Go to **System > Remote Desktop**
3. Turn ON **Remote Desktop**
4. Click **Confirm**

💡 Note: Windows Home cannot host RDP

---

## 🌐 Step 2: Find Your IP Address

1. Press `Win + R`
2. Type: `cmd`
3. Run: `ipconfig`
4. Look for **IPv4 Address**

---

## 🖥️ Step 3: Open Remote Desktop Connection

1. Press `Win + R`
2. Type: `mstsc`
3. Press Enter

---

## 🔑 Step 4: Connect

1. Enter the IP address
2. Click Connect
3. Enter username and password

---

## 🔒 Security Tips

- Use strong passwords
- Enable firewall
- Avoid exposing port 3389
- Use VPN

---

## ❗ Troubleshooting

- Can't connect → Check IP & RDP enabled  
- Timeout → Check firewall & port 3389  
- Login failed → Check credentials  

---

## ✅ Done!

You can now use RDP on Windows 🎉
